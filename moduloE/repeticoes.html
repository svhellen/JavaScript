Repetições

While - Estrutura de repetição com teste lógico/condição no início

    
    var c = 1 
    while (c <= 6) {
        console.log(`Passo ${c}`)
        c++
    }

    Saída:
    Passo 1
    Passo 2
    Passo 3
    Passo 4
    Passo 5
    Passo 6

Do While - Estrutura de repetição com teste lógico/condição no final

    
    var c = 1 
    do {
        console.log(`Passo ${c}`)
        c++
    } while (c <= 6) 

    Saída:
    Passo 1
    Passo 2
    Passo 3
    Passo 4
    Passo 5
    Passo 6

For - Estrutura de repetição com variável de controle

    
        
    for (var c = 1; c <= 6; c++) {
        console.log(`Passo ${c}`)
    }

    Saída:
    Passo 1
    Passo 2
    Passo 3
    Passo 4
    Passo 5
    Passo 6
