    Atribuição

    Atribuição simples

  • = Recebe

    • Exemplos:

    • var a = 5 + 3
    • var b = a % 5
    • var c = 5 * b ** 2
    • var d = 10 - a / 2
    • var e = 6 * 2 / d
    • var f = b % e + 4 / e

    Auto-atribuição

    Exemplos:

    • var n = 3
    • n = n + 4 ou n+=4
    • n = n - 5 ou n-=5
    • n = n * 4 ou n*=4
    • n = n / 2 ou n/=2
    • n = n ** 2 ou n**=2
    • n = n % 5 ou n%=5

    Incremento

  • x++ (x = x + 1)
  • x-- (x = x - 1)

    • Relacionais

    Para toda a expressão que tenha um operador relacional o resultado sempre será booleano (verdadeiro ou falso).

    • < menor que
    • > maior que
    • <= menor ou igual a
    • >= maior ou igual a
    • == igual a
    • != nao igual a ou diferente de
    • === idêntico a

    Lógicos

    • ! negação, 'não'
    • && conjunção, 'e'
    • || disjunção, 'ou'

    Ternário

    O operador ternário faz a junção de três operandos:
    Teste ? True : False

  • ? interrogação
  • : dois pontos

    • Exemplo:

    média >= 7.0 ? "Aprovado" : "Reprovado"
